# Zomato Performance Analytics Dashboard

This project is an end-to-end Business Intelligence solution developed using Power BI, SQL, Excel, DAX, and Power Query.

The dashboard provides comprehensive insights into Zomato's business performance by analyzing sales trends, customer behavior, restaurant performance, food categories, ratings, and city-level analytics.

## Project Objectives

- Analyze sales performance across cities and restaurants
- Track customer acquisition and retention
- Monitor active users and order trends
- Identify top-performing cities and restaurants
- Analyze food category performance
- Generate business insights through interactive dashboards

## Tools & Technologies

- Power BI
- SQL
- Excel
- DAX
- Power Query

## Data Model

The project uses 5 interconnected tables:

1. Orders
2. Users
3. Restaurant
4. Food
5. Menu

A relational data model was created to support efficient analysis and reporting.

## Dashboard Pages

### 1. Executive Dashboard
- Revenue Analysis
- Order Analysis
- Rating Analysis
- Top Performing Cities
- Sales Trends

### 2. User Performance Dashboard
- Active Users
- Gain Customers
- Lost Customers
- Average Rating
- User Demographics

### 3. City Analysis Dashboard
- Revenue by City
- Rating by City
- Active User Rate
- Top Rated Cities
- Geographic Analysis

## Key KPIs

- Total Revenue
- Total Orders
- Total Quantity Sold
- Active Users
- Average Rating
- Gain Customers
- Lost Customers
- City Performance Metrics

## Business Impact

The dashboard enables stakeholders to make data-driven decisions by identifying customer trends, city performance, restaurant effectiveness, and revenue opportunities.
